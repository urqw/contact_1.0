# Contact 1.0: Invasion

Text-based game "Contact 1.0: Invasion" (Russian: "Контакт 1.0: Вторжение") by Chicago1920.

The game is in Russian, developed for AkURQ, and adapted for [UrqW](https://github.com/urqw/UrqW) text-based game engine.

This repository contains the source code and other resources of the game. The game is available for launch in the [UrqW engine online catalog](https://urqw.github.io/UrqW/#contact_1.0).

The game was first published on February 22, 2010.

Interactive fiction identifier (IFID) of the game which is its MD5 hash: E5AFCE4671BACF97757223F0411E89CA

## Description

Не далёкое будущее. Или близкое прошлое... Но однажды произошёл контакт. Корабли пришельцев прилетели на Землю. Явно, что намерения их были совсем не дружелюбными... Нам придётся играть в роли простого солдата секретной части. Несколько классов монстров. Оружие на любой вкус. Загадки и многое другое.
